TROUBLESHOOTING
================================================================================

Icons Not Showing?
   → npm run build:icons
   → npm run dev
   → Check build/ directory has icon files

Icon Looks Blurry?
   → At 16×16: Design may have too much detail
   → Simplify SVG if needed
   → Regenerate: npm run build:icons

Windows Icon Missing?
   → Verify: ls build/icon.ico
   → Regenerate: npm run build:icons
   → Or download from online converter

macOS Icon Missing?
   → On macOS: npm run build:icons creates .icns
   → Not on macOS: Create manually with iconutil
   → See ICON_SETUP.md for instructions

Favicon Not Updating?
   → Hard refresh: Cmd+Shift+R (macOS) or Ctrl+Shift+R (Windows)
   → Clear browser cache completely

For More Help:
   → See SETUP_ICONS.md for quick troubleshooting
   → See ICON_SETUP.md for detailed help
   → See LOGO_VISUAL_GUIDE.txt for diagrams
