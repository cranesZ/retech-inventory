CUSTOMIZATION POINTS
================================================================================

To modify the logo, edit public/logo.svg:

Color Changes:
    Find: <stop offset="0%" style="stop-color:#0066CC
    Change: #0066CC to your color code
    Regenerate: npm run build:icons

Element Positioning:
    Find: <rect x="80" y="140" ...  (shelf positions)
    Change: x and y coordinates (0-512 range)
    Regenerate: npm run build:icons

Add/Remove Elements:
    Find: <g id="..."> sections
    Edit: Comment out or remove groups
    Regenerate: npm run build:icons

Text Modification:
    Find: <text> ... </text> sections
    Change: Text content or positioning
    Regenerate: npm run build:icons

TROUBLESHOOTING FLOWCHART
================================================================================

Icon not showing in app?
    │
    ├─→ Run: npm run build:icons
    │   └─→ Check: ls build/icon*
    │       └─→ Continue
    │
    └─→ Rebuild: npm run build
        └─→ Restart: npm run dev

Icon looks blurry?
    │
    ├─→ At small size (16×16)?
    │   └─→ SVG may have too much detail
    │       └─→ Simplify and regenerate
    │
    └─→ At large size (512×512)?
        └─→ PNG compression too aggressive
            └─→ Regenerate with quality settings

Icon wrong on Windows?
    │
    ├─→ Check: ls build/icon.ico
    │   └─→ Doesn't exist? Generate or download
    │
    └─→ Test: Open icon.ico in viewer
        └─→ If corrupt, regenerate

Icon missing on macOS?
    │
    ├─→ Check: ls build/icon.icns
    │   └─→ Doesn't exist and not on macOS?
    │       └─→ Create manually: iconutil -c icns icon.iconset
    │
    └─→ If exists, rebuild: npm run build

DESIGN PHILOSOPHY

✓ RECOGNIZABLE
